## 📝 设计决策记录

### 2024-XX-XX: 架构选择
**决策**: 采用 MVVM 架构模式  
**原因**: 
- 清晰的职责分离
- 易于测试和维护
- 符合Flutter最佳实践

### 2024-XX-XX: 状态管理选择
**决策**: 使用 Provider + ChangeNotifier  
**原因**:
- 官方推荐
- 学习曲线平缓
- 适合中小型项目

### 2024-XX-XX: 数据存储选择
**决策**: SQLite (sqflite) + SharedPreferences  
**原因**:
- SQLite适合结构化数据存储
- SharedPreferences适合简单配置
- 无需云端依赖

# Memo - 功能特性详解

本文档详细介绍 Memo 应用的所有功能特性和使用方法。

---

## 📱 应用概览

Memo 是一款集打卡、番茄钟和待办事项管理于一体的效率管理应用，采用 Material Design 3 设计风格，提供简洁优雅的用户体验。

### 核心模块

1. **首页** - 今日概览和快速统计
2. **番茄钟** - 专注计时和时间管理
3. **日历** - 历史记录可视化
4. **待办事项** - 任务管理和提醒
5. **打卡** - 习惯养成和记录（通过底部导航切换）
6. **设置** - 个性化配置

---

## 🏠 首页 - 今日概览

### 功能特性

#### 智能问候
- 根据时间自动显示问候语
  - 早上好（00:00 - 11:59）
  - 下午好（12:00 - 17:59）
  - 晚上好（18:00 - 23:59）

#### 今日概览卡片
显示当天的核心数据：
- **打卡状态**: 已完成/未完成
- **连续天数**: 连续打卡的天数统计
- **番茄钟**: 今日完成的番茄钟数量

#### 打卡状态卡片
- 显示今日打卡状态
- 连续打卡天数激励
- 快速跳转到打卡页面的入口

#### 番茄钟统计卡片
- 今日完成的番茄钟数量
- 今日专注时长（分钟）
- 番茄钟运行状态指示
- 快速查看当前是否有番茄钟运行中

### 使用场景
- 每天打开应用时快速了解今日进度
- 查看个人效率数据
- 激励自己保持良好习惯

## ⏰ 番茄钟 - 专注计时

### 功能特性

#### 时钟选择器
- **可视化时钟盘**: 拖动选择时间
- **数字时间显示**: 实时显示选中的时间
- **快捷按钮**: 
  - OK：确认并开始倒计时
  - Cancel：取消选择

#### 倒计时功能
- **大号数字显示**: 清晰可见的倒计时
- **进度圆环**: 可视化进度展示
- **控制按钮**:
  - 暂停/继续
  - 取消（需确认）

#### 专注模式
- **Tab 锁定**: 番茄钟运行时锁定底部导航
- **视觉提示**: 导航栏变灰显示锁定状态
- **专注保护**: 防止误操作打断专注状态

#### 倒计时结束
- **通知提醒**: 系统通知提醒倒计时结束
- **自动记录**: 完成的番茄钟自动保存到数据库
- **统计更新**: 实时更新今日统计数据

### 使用方法

1. **开始番茄钟**
   - 打开番茄钟页面
   - 在时钟盘上选择时长（默认 25 分钟）
   - 点击 OK 开始倒计时

2. **暂停/继续**
   - 点击暂停按钮暂停倒计时
   - 再次点击继续倒计时

3. **取消倒计时**
   - 点击 Cancel 按钮
   - 确认取消操作
   - 倒计时记录不会保存

4. **专注状态**
   - 倒计时运行时底部导航被锁定
   - 无法切换到其他页面
   - 专注于当前任务

### 使用场景
- 工作学习时保持专注
- 时间管理（25 分钟工作法）
- 提高工作效率
- 记录每日专注时长

## 📅 日历 - 历史记录

1. **查看月份**
   - 左右箭头切换月份
   - 查看历史记录

2. **查看详情**

   - 查看当天的打卡和番茄钟记录

### 使用场景

- 统计月度数据

## ✅ 待办事项 - 任务管理

### 功能特性

#### 任务管理
- **添加待办**: 快速添加新任务
- **编辑待办**: 修改任务详情
- **删除待办**: 滑动删除或编辑删除
- **完成切换**: 一键标记完成/未完成

#### 任务属性
- **标题**: 必填，最多 100 字符
- **描述**: 可选，最多 500 字符
- **优先级**: 低/中/高（蓝/橙/红）
- **到期日期**: 可选，日期选择器

#### 智能过滤
- **全部**: 显示所有未完成任务
- **今天到期**: 只显示今天到期的任务
- **已逾期**: 显示已过期但未完成的任务
- **高优先级**: 只显示高优先级任务

#### 多种排序
- **按优先级**: 高→低，相同优先级按到期日期
- **按到期日期**: 近→远，无日期的排最后
- **按创建日期**: 新→旧

#### 状态分组
- **未完成 Tab**: 显示所有未完成任务
- **已完成 Tab**: 显示所有已完成任务
  - 可批量清空已完成

#### 视觉提示
- **逾期任务**: 红色警告图标和文字
- **今天到期**: 橙色提示图标
- **优先级标签**: 彩色标签（低/中/高）

#### 交互操作
- **滑动删除**: 向左滑动快速删除
- **点击复选框**: 切换完成状态
- **点击编辑图标**: 编辑任务详情

### 使用方法

1. **添加待办事项**
   - 点击右下角 "+" 按钮
   - 输入标题（必填）
   - 选择性添加：
     - 详细描述
     - 到期日期
     - 优先级（默认中）
   - 点击"保存"

2. **编辑待办事项**
   - 点击待办项右侧编辑图标
   - 修改任意字段
   - 点击"保存"

3. **删除待办事项**
   - 方法1：向左滑动待办项
   - 方法2：编辑时在对话框中删除

4. **标记完成**
   - 点击待办项左侧圆形复选框
   - 自动移动到"已完成" Tab

5. **过滤和排序**
   - 点击顶部过滤图标选择过滤条件
   - 点击排序图标选择排序方式

6. **清空已完成**
   - 切换到"已完成" Tab
   - 点击"清空已完成"按钮
   - 确认删除

### 使用场景
- 日常工作任务管理
- 学习计划安排
- 生活待办清单
- 项目任务跟踪
- 目标管理

## 📍 打卡 - 习惯养成

### 功能特性

#### 统计卡片
- **连续打卡**: 显示连续打卡天数（橙色火焰图标）
- **总打卡数**: 显示累计打卡天数（绿色勾选图标）

#### 打卡功能
- **每日打卡**: 每天可打卡一次
- **打卡备注**: 可选添加备注（最多 200 字符）
- **打卡时间**: 自动记录打卡时间
- **打卡状态**: 大号绿色勾选图标

#### 打卡历史
- 显示最近的打卡记录
- 包含日期、时间和备注
- 列表展示，易于查看

#### 视觉反馈
- 未打卡：绿色打卡按钮
- 已打卡：绿色勾选标识
- 成功提示：绿色 SnackBar

### 使用方法

1. **每日打卡**
   - 打开打卡页面
   - 点击大号打卡按钮
   - 可选输入备注
   - 点击"打卡"

2. **查看历史**
   - 下拉刷新数据
   - 查看打卡历史记录
   - 查看连续打卡天数

### 使用场景
- 养成每日习惯
- 记录学习/工作打卡
- 坚持目标追踪
- 激励自己保持连续

## 🔐 数据安全

### 本地存储
- 所有数据存储在本地 SQLite 数据库
- 不上传到云端
- 隐私安全有保障

### 数据持久化
- 自动保存所有操作
- 应用关闭后数据不丢失
- 数据库自动备份

## 📝 常见问题

### Q1: 番茄钟可以自定义时长吗？
A: 可以。在设置页面中可以设置默认番茄钟时长（5-120分钟）。

### Q2: 打卡每天可以打几次？
A: 每天只能打卡一次，鼓励养成每日习惯。

### Q3: 待办事项会自动提醒吗？
A: 当前版本通过逾期标记提醒，暂不支持推送通知。

### Q4: 数据会丢失吗？
A: 所有数据存储在本地数据库，不会丢失。但卸载应用会清空数据。

### Q5: 支持数据导出吗？
A: 当前版本暂不支持，将在未来版本中添加。
